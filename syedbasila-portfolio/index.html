Project Details

"ASL Interpreter Robot : I led the development of an ASL Interpreter Robot, designed to translate audio and text into American Sign Language (ASL) and vice versa. The project utilized Python, OpenCV, and Arduino UNO, along with embedded C for hardware control. The robot interprets spoken language into ASL gestures using a robotic hand, while also converting ASL gestures back into text and audio. This innovative solution aims to bridge communication gaps for the deaf and blind communities, making interactions more accessible and inclusive."
